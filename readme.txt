Hercules Post Stats adds a meta box to your post edit screen that shows you real time stats about your post.

== Description ==

Hercules Post Stats adds a meta box to your post edit screen that shows you real time details about
how many words you have written in the post, how many you wrote this session, how much time total you
have spent on the post, how long you have spent this session, and approximately how long it would take
the average reader to read the post.

== Installation ==

Add this plugin by uploading the zip using the "Add Plugin" feature built into Wordpress.  Otherwise manaully unzip the folder
and upload the entire directory to your blog's plugins folder ( /wp-content/plugins/ ).  Next you have to activate the plugin.

Alternatively you can install it using the Wordpress plugin search from the plugins page in your Wordpress admin area.

== Frequently Asked Questions ==

= What do these stats do for me? =

These stats are useful for blog writers when they are writing to know how productive they are being.  It shows how many words you have
written in this writing session, and how much time you have spent while also showing the total words and total writing time.  In addition
it shows you how long it would take the average reader to read the post.
